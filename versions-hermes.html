Conclusion courte

La première version ressemble à un agent Python déjà utilisable; la version 17/dernière est devenue un produit multi-interface: agent + gateway messagerie + WebUI/desktop + TUI + plugins/MCP + cron + profils + catalogue de skills + packaging. Les plus gros changements ne sont pas seulement “des fichiers ajoutés”, mais une transformation vers une plateforme exploitable sur VPS, desktop et messageries.
