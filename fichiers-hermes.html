Par où commencer selon ton objectif

Ajouter une capacité

  1. Créer un fichier dans tools/.
  2. Enregistrer le tool dans tools/registry.py.
  3. L’ajouter au bon toolset dans toolsets.py.
  4. Tester avec la CLI/gateway selon l’usage.

Comprendre une réponse Telegram

  1. gateway/platforms/telegram.py reçoit le message.
  2. gateway/run.py route vers l’agent.
  3. agent/prompt_builder.py prépare le contexte.
  4. run_agent.py appelle le modèle et tools.
